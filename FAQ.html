Faq


Do I need to obtain an API Key to get started playing with the GIPHY SDK? Yes. You can obtain an individual rate-limited key by logging into this developer portal with your GIPHY account and then creating an app.

I’ve been approved for a production key! Do I get a new API key? No, your production key is the same API key that you created in your dashboard, with upgraded permissions.

What is the beta key? When you create an app, you will receive a rate limited beta key, which we advise you to use in development.

How do I upgrade my beta key to production? Click on the “Request a production key” link on the app dashboard. We’ll ask you to provide a little bit more info about your app.
